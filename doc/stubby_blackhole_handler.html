init/2

init(Req0::cowboy_req:req(), State::any()) -> {ok, cowboy_req:req(), any()}

A request recording handler mounted to stubby server at /blackhole/[...]. When requested, the request body is enqueued to recorder FIFO queue, and a response with status code 204 is returned. A request body can be gzipped when Content-Encoding: gzip is specified.
